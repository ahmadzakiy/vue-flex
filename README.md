# vue-flex
### Vue single file component for Flex layout

- Based by Vue-Cli 3.0

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Build for Library
```
npm run build:lib
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
